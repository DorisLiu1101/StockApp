# 📈 維股利 (StockApp) 

一款專為個人投資者打造的漸進式網頁應用程式 (PWA)。
系統採用**企業級模組化架構**，結合 Firebase 即時資料庫與 Google Apps Script (GAS) 後端，並獨家導入 **Gemini AI** 進行多維度個股分析與動態估值，全面支援台股 (TW)、美股 (US) 與日股 (JP) 市場。

---

## 🌟 核心功能 (Key Features)

* **📊 庫存總覽**：即時資產計算、動態損益追蹤與產業圓餅圖分佈。
* **⭐ 觀察名單**：獨立的收藏清單與專屬極簡資訊視窗，隨時掌握潛力標的。
* **🤖 AI 聯網報告**：一鍵生成巴菲特五維度體質分析、即時財報掃描與三情境風險矩陣。
* **🎯 估值儀表板**：內建雙軌 PE 模型 (絕對倍數 / 歷史動態)，自動計算淑價與貴價。
* **📱 原生級體驗**：採用 History API 攔截路由，具備防跳出機制與流暢的單頁式 (SPA) 切換體驗。

## 📂 專案檔案結構 (File Architecture)

目前專案已全面升級為模組化架構，將 UI 介面與商業邏輯徹底分離：

```text
/
├── index.html                 # 主程式進入點 (極簡 UI 介面、彈窗骨架、底部導覽列)
├── StockApp_Report_Form.html  # AI 報告專屬渲染模板 (估值儀表板、雷達分析 UI)
├── manifest.json              # PWA 設定檔 (定義 App 名稱、主題色、全螢幕啟動模式)
├── icon.png                   # App 桌面圖示 (Logo)
└── /js                        # 核心邏輯模組 (Business Logic)
    ├── app.js                 # 系統大腦：Firebase 連線、登入狀態、全域同步與路由攔截
    ├── portfolio.js           # 庫存模組：庫存清單渲染、圓餅圖計算、庫存詳情視窗
    ├── pick.js                # 收藏模組：觀察名單過濾、動態排序、收藏專屬視窗
    ├── tx.js                  # 交易模組：新增買賣紀錄、表單防呆、後端寫入對接
    └── report.js              # AI 報告模組：串接 Gemini API 進行聯網分析與估值運算
